Coding Coaches API

Welcome to the Coding Coaches RESTful API

The API serves as the back-end for the Coding Coaches Angular app. Both projects are starter templates and demonsttration code portfolios by me, Ryan Hunter as part of my ongoing coding practice. The people / "coaches" in the site are fictional, and are largely based on the excellent free data from the RamdomUser.me site.

API : /users
To access and modify resources, you can use any HTTP method:
GET POST PUT PATCH DELETE OPTIONS
